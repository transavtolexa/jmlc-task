# Восстановление пробелов в поисковых запросах

Сервис для восстановления пробелов в слитно написанных русскоязычных поисковых запросах на платформе Авито.

Проект принимает строку без пробелов и возвращает более читаемый вариант с восстановленными границами слов. Для решения используется модель на уровне символов, а результат можно получить через FastAPI-интерфейс или через командную строку.

Примеры:

```text
куплюайфон14про -> куплю айфон 14 про
ищудомвПодмосковье -> ищу дом в Подмосковье
```

## Установка

Для запуска нужен Python 3.9-3.12 и Poetry.

Если Poetry еще не установлен, на macOS его можно поставить через Homebrew:

```bash
brew install poetry
```

Если Homebrew не используется, Poetry можно установить через Python:

```bash
python3 -m pip install poetry
```

Затем установите зависимости проекта:

```bash
poetry install
```

## Запуск API

```bash
poetry run python -m uvicorn app.main:app --reload
```
Откройте веб-страницу:

```text
http://127.0.0.1:8000/
```

## Запуск через CLI

```bash
poetry run python -m app.segmenter "куплюайфон14про"
```

## Если что-то не запускается

Если терминал пишет `poetry: command not found`, проверьте, что Poetry установлен и доступен в `PATH`.

Если терминал пишет `Command not found: uvicorn`, сначала выполните:

```bash
poetry install
```
